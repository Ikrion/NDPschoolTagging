Third-Party Services

Our application uses Google Cloud Platform, Google APIs, and Amazon Web Services (AWS) to provide and support its functionality.

Google services are used for authentication and accessing authorized Google Sheets data with user consent.

AWS is used for backend processing (AWS Lambda), user authentication (AWS Cognito), and data storage (Amazon S3 and DynamoDB).

AWS services used include:

  • AWS Cognito for user authentication and identity management (including Google sign-in integration)
  • AWS Lambda for backend processing and data transformation
  • Amazon S3 for storing processed application data
  • Amazon DynamoDB for storing user settings and lightweight application state data

Your use of Google services is subject to Google's privacy policies and terms. Your use of AWS services is subject to Amazon Web Services' privacy policies and terms.

Data Storage and Processing

With user authorization, data retrieved from Google Sheets is processed using AWS Lambda functions, where it is filtered, transformed, and converted into structured JSON format.

Processed data may be stored in Amazon S3 for operational processing and system workflows.

User-specific settings and lightweight application state information may be stored in Amazon DynamoDB to enable faster access and improve application performance.

All data processing and storage are performed solely to support application functionality, and are not used for advertising, profiling, or marketing purposes.
